# Vehicle-Price-Prediction-Rusty-Bargain-
### Type: 
Regression / Model Comparison
### Tools: 
Python, pandas, NumPy, scikit-learn, LightGBM, CatBoost

Decision Tree
Random Forest
Gradient Boosting

Tuned hyperparameters
Compared speed vs accuracy

## Outcome:

Identified best model for scalable predictions
Balanced performance and efficiency
